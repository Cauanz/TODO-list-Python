# TODO list Python

It's a classic and very simple TODO list project made with Python for task management through the terminal.

## Installation

Just clone the project to your local machine.

```bash
git clone
```

## Usage

```python
# starts the program
py exec.py

====== O que deseja fazer: ======
|  1 - Adicionar nova tarefa    |
|  2 - Ver tarefas              |
|  3 - Atualizar tarefa         |
|  4 - Remover tarefa           |
|  5 - Sair                     |
=================================

# 1 - Adds a task
# 2 - Shows all tasks
# 3 - Update a task
# 4 - Remove a task
# 5 - Exits the program
```
